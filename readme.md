# Landing Page Multiverso Spider-Man | HTML CSS JS


Landing Page do multiverso Spider-Man, aborda todos os filmes do universo aranha.

Desenvolvida em HTML, CSS, JS, a página possui cards com animações, que se rotacionam a cada clique.
